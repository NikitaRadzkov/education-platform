# First Steps

## Short Description

This theory provides basic knowledges about how to setup python, basic syntax and package management

## Topics

* Language fundamentals
  * Data types
    * Basic types
    * Mutability
    * Iterable basic types
    * List vs Tuple
  * Comprehensions
    * List comprehensions
    * Dict comprehensions
    * Generator comprehensions
    * Iterators and iterable objects
  * Env management
    * virtualenv
    * package managers
      * pip
* Built-ins
  * filter
  * map/reduce
