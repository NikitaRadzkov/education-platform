# Intermediate topics

## Short Description

This theory provides knowledge about some frequently used language features and tecniques

## Topics

* Language fundamentals
  * Scope of variables (LEGB)
    * Global
    * Local
    * Enclosed
    * Built-in
  * Exceptions
* Functions
  * Decorators
  * Generator functions
  * Closures
* Built-ins
  * filter
  * map/reduce
  * dir
  * pickle
  * os
* File IO
  * open, close
  * context manager
  * modes(read, write, binary, append)
