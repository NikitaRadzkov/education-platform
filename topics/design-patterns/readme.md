## Introduction

This course is intended for giving you an overview of the most commonly
used design patterns.

## Topics

* Design patterns
  * Creational patterns
    * Factory method
    * Abstract factory
    * Builder
    * Prototype
    * Singleton
  * Structural patterns
    * Adapter
    * Bridge
    * Composite
    * Decorator
    * Facade
    * Flyweight
    * Proxy
  * Behavioral patterns
    * Chain of responsibility
    * Command
    * Memento
    * Observer
    * Template method
    * Visitor
    * Iterator
    * Mediator
    * State
    * Strategy
