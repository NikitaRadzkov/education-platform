# Own Image

## Short Description

Build your own Dockerfile and Run container.

## Estimation (h)

6

## Requirements

1. Take an existing app and Dockerize it.
2. Make Dockerfile.
3. Build it.
4. Test it.
5. Push it.
6. Run it.
7. Tag and push to your Docker Hub account.
8. Remove your image from local cache, run again from Hub.
