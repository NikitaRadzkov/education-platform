# Java Fundamentals

## Version

1.0.0

## Introduction

This container will allow you to study Java basics with minimum of OOP.
You will get familiar with Java fundamentals and will learn how to use it for writing basic programs.

## Topics

### Beginner

* OOP
  * Classes and Objects
  * Environment
  * Packages
  * Interfaces
  * Inheritance
* Java language
  * Variables
  * Data types
  * Numeric data types
  * Characters
  * Strings
  * Operators
  * Expressions
  * Loops and conditions
* Collection Framework
  * Collection Interfaces
  * Collection Implementations
* Java library
  * Math Classes

### Intermediate

* Java language
  * Generics
  * Exception Handling
  * Annotations
* Java library
  * String Buffer and Builder
  * Regular Expressions
  * Java Date and Time
* Logging
* Streams and Threading
  * Aggregate Operations and Stream API
  * Java IO, NIO and NIO.2
  * Java File IO
  * Multi-threading and Synchronization

### Advanced

* Java language
  * Javadoc
* Java library
  * Serialization and De-Serialization
  * Reflection
  * JDBC
  * JMX
  * Networking

## Prerequisites

### Environment

* Intellij IDEA
